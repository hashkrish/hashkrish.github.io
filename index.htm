Projects Handled

Hinged Span UAV

The project is to create a UAV with hinged Span support. Why Hinged Span? It is not easy to produce a sudden vertical downward movement on aircrafts. The hinged wings are used to produce the sudden down fall of the UAV by Closing the span.

Continuous Speed Varying Pulleys

In this work, variable speed power transmission system has been designed for the specified speed ratio range from 0.4 to 2.5. This system consists of taper pulleys, belt drive and fork. Here the fork is used to change the position of the belt.From the CVT system the continuous power transmission for the specified speed ratio is achieved from the pair of taper pulleys.

Mobile Operated Remote Module

An arduino based project to operate a remote module through bluetooth module and smartphone. The smartphone sends data as numbers such as 1,2,3,... over bluetooth which is used to navigate the remote module. A simple project with the few lines of code.
